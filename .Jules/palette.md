# Palette's Journal

## 2026-01-05 - Search Experience
**Learning:** The "Clear Search" pattern is highly effective in single-input search pages. Using `AnimatePresence` for swapping the search icon with the clear button creates a delightful micro-interaction that feels polished.
**Action:** Always consider how a user will *reset* or *change* their input, not just how they enter it. Add clear buttons to all major search inputs in the future.
